Commands

Moderation

  • /kick [user] [reason] - Kick a user
  • /ban [user] [reason] - Ban a user
  • /nick [user] [nickname] - Change a user’s nickname
  • /purge [amount] - Delete recent messages
  • /hackban [user_id] [reason] - Ban by user ID
  • /archive [limit] - Archive last messages
  • /mute [user] [duration] [reason] - Mute a user
  • /unmute [user] [reason] - Unmute a user
  • /warning add [user] [reason] - Add a warning
  • /warning remove [user] [warn_id] - Remove a warning
  • /warning list [user] - List warnings

Music

  • /play [song name or YouTube URL] - Play a song from YouTube
  • /pause - Pause the currently playing track
  • /resume - Resume the paused track
  • /skip - Skip the current song (only if there are songs in the queue)
  • /queue - Show the current music queue
  • /leave - Stop playback and leave the voice channel

General

  • /help - List all commands the bot has loaded
  • /botinfo - Get information about the bot
  • /serverinfo - Get information about the server
  • /ping - Check if the bot is alive
  • /invite - Get the invite link for the bot
  • /server - Get the invite link for the support server
  • /8ball [question] - Ask any question to the bot
  • /feedback - Submit feedback for the owners of the bot

Leveling

  • /rank [user] - Show a users XP and level
  • /leaderboard - Show the server XP and level leaderboard
  • /levelrole [level] [role] - Set a role to be given at a certain level (admin only)
  • /levelset [user] [level] - Set a user's level (admin only)
  • /levelsettings [message] [channel] - Configure the leveling system (admin only)

Setup

  • /setup-moderation - Start the moderation setup process for roles and permissions (admin only)
  • /setup-logs - Setup your server's logging system and select which events to log (admin only)
  • /setup-ticket - Setup the ticket system for your server (admin only)
